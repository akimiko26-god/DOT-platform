# Публикация ./dot в интернет (бесплатно)

## Вариант 1: Быстро с этого ПК (Cloudflare Tunnel)

Не нужен домен и проброс портов. Пока открыто окно туннеля — сайт доступен из интернета.

```powershell
cd C:\Users\Admin\dot-platform
.\start-public.ps1
```

В консоли появится ссылка вида `https://xxxx.trycloudflare.com` — откройте её в браузере на любом устройстве.

**Минусы:** компьютер должен быть включён; при каждом запуске URL может меняться (у quick tunnel).

---

## Вариант 2: Постоянный хостинг Render.com (free tier)

1. Залейте проект на GitHub.
2. Зарегистрируйтесь на [render.com](https://render.com).
3. **New → Blueprint** → выберите репозиторий (файл `render.yaml` подхватится).
4. После деплоя откройте URL вида `https://dot-platform-xxxx.onrender.com`.

**Минусы free tier:** сервис «засыпает» без посещений (~1 мин на пробуждение); SQLite и загрузки могут сброситься при пересборке.

Переменные окружения на Render (по желанию): `SECRET_KEY`, SMTP для почты сброса пароля.

`FRONTEND_URL` подставится автоматически из `RENDER_EXTERNAL_URL`.
